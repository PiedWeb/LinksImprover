## Support this package

## Installation

You can install the package via composer:

```bash
composer require piedweb/LinksImprover
```

## Usage

``` php
$skeleton = new Piedweb\LinksImprover();
echo $skeleton->echoPhrase('Hello, Piedweb!');
```
